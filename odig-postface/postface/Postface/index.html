Introduction

While working on YOCaml, XHTMLBoy told us that the interaction between the (deliberately small) standard Preface library and the standard OCaml library could be irritating.

As it is great fun to reinvent the wheel, we decided to build a standard library (yes, another one) that would implement the abstractions proposed by Preface. The aim is that as soon as someone (original contributor or not) finds an interesting and highly reusable function, they add it to the library (with clear documentation and, obviously, unit tests).
