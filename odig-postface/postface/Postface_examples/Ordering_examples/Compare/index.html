Module Ordering_examples.Compare

Examples for Postface.Ordering.compare

Ordering respects this order: Lt < Eq < Gt. So:

# Ordering.(compare Eq Eq) ;;
- : Ordering.t = Eq
# Ordering.(compare Lt Eq) ;;
- : Ordering.t = Lt
# Ordering.(compare Gt Lt) ;;
- : Ordering.t = Gt
