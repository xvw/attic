Module List_examples.Length

Examples for Postface.List.length

length returns the size of a list. The whole list must be traversed to know the size.

# List.length [] ;;
- : int = 0
# List.length [1; 2; 3] ;;
- : int = 3
# List.length (10 -- 22021) ;;
- : int = 22012
