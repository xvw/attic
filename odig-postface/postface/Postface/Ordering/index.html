Introduction

The Standard OCaml library uses integers to describe the results of comparisons (if the integer is equal to 0, the objects being compared are equal, if the integer is greater than 0, the first object being compared is larger than the second, and if the integer is less than 0, the first object being compared is smaller than the second), Postface took the decision to use a closed enumeration to reduce the domain of comparison functions.
