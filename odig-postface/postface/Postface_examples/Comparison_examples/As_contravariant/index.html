Module Comparison_examples.As_contravariant

Examples of using Comparison as a Postface.Comparison.Contravariant Functor

As for the use of predicates with a contravariant functor, using contramap (or its infixed version >$<) changes the focus of the comparison application. If we wanted to write a compare_length equivalence we could write it this way:

# let compare_length l1 l2 = (List.length >$< cmp) l1 l2;;
val compare_length : 'a list -> 'a list -> Ordering.t = <comparison>

The list sizes are compared:

# compare_length [] [] ;;
- : Ordering.t = Eq
# compare_length [1; 2] [] ;;
- : Ordering.t = Gt

As with predicates, contramap will first apply a function to the observed subject whose result will be applied to the comparison passed as an argument.
