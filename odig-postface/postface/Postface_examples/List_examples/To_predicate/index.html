Module List_examples.To_predicate

Examples for Postface.List.has_length, Postface.List.is_larger and Postface.List.is_smaller

Comparisons can be easily lifted into predicates.

# List.has_length 3 [1; 2; 3] ;;
- : bool = true
# List.is_larger 3 [1] ;;
- : bool = false
# List.is_smaller 10 ["OCaml"] ;;
- : bool = true
